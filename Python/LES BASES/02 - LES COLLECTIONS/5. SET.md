⚠️ C'est une collection **non ordonnée** et qui **n'accepte pas les doublons**. Il s'écrit avec des **accolades** `{ }` mais **sans clés** (contrairement au dict)

➡️ Pas d'index — on ne peut pas faire `set[0]`
➡️ Les doublons sont **automatiquement supprimés** 
➡️ Très utile pour **dédoublonner** une liste ou faire des **comparaisons entre groupes**

[plop]('Python/LES BASES/02 - LES COLLECTIONS')

#### 💡 <span style="color: cyan;">Exemple</span>

```python
animaux = {"chat", "chien", "lapin", "chat", "chien"}
print(animaux)   # → {"chat", "chien", "lapin"}  les doublons ont disparu !
```

⚠️ Piège visuel — dict ou set ?
```python
mon_dict = {"nom": "Sara"}   # dict  → clé : valeur
mon_set  = {"Sara", "Lena"}  # set   → juste des valeurs
mon_set_vide = set()         # ✅ set vide
mon_dict_vide = {}           # ⚠️ Python voit un dict vide, pas un set !
```

### 📦 Ajouter / Supprimer

⚠️ Ajouter un élément
```python
animaux = {"chat", "chien", "lapin"}

animaux.add("perroquet")
print(animaux)   # → {"chat", "chien", "lapin", "perroquet"}

animaux.add("chat")   # déjà présent → rien ne se passe, pas d'erreur
print(animaux)   # → {"chat", "chien", "lapin", "perroquet"}
```

⚠️ Supprimer un élément
```python
animaux.remove("chien")    # ❌ erreur si l'élément n'existe pas
animaux.discard("chien")   # ✅ pas d'erreur si l'élément n'existe pas
animaux.pop()              # supprime un élément au hasard (ordre imprévisible !)
animaux.clear()            # vide le set
```

### 📦 Chercher dans un set

⚠️ Taille et existence
```python
animaux = {"chat", "chien", "lapin"}

print(len(animaux))          # → 3
print("chat" in animaux)     # → True
print("perroquet" in animaux) # → False
```

### 📦 Les opérations entre deux sets 

⚠️ C'est le **super pouvoir** des sets — comparer deux groupes facilement

##### 💡 <span style="color: cyan;">Exemple</span>

```python
aimé_sara = {"python", "musique", "sport"}
aimé_lena = {"python", "cuisine", "sport"}
```

⚠️ Union — tout ce qui est dans l'un **ou** l'autre
```python
print(aimé_sara | aimé_lena)
# → {"python", "musique", "sport", "cuisine"}
```

⚠️ Intersection — ce qui est dans l'un **et** l'autre
```python
print(aimé_sara & aimé_lena)
# → {"python", "sport"}
```

⚠️ Différence — ce qui est dans sara **mais pas** dans lena
```python
print(aimé_sara - aimé_lena)
# → {"musique"}
```

⚠️ Différence symétrique — ce qui est dans l'un **mais pas dans les deux**
```python
print(aimé_sara ^ aimé_lena)
# → {"musique", "cuisine"}
```

### 📦 Dédoublonner une liste

⚠️ Le set est parfait pour enlever les doublons d'une liste en une ligne

```python
notes = [15, 18, 15, 12, 18, 9, 15]
notes_uniques = list(set(notes))
print(notes_uniques)   # → [9, 12, 15, 18]  plus de doublons !
```

### 🔍 RECAP

```
.add(x)         → ajoute x (sans doublon)
.remove(x)      → supprime x, erreur si absent
.discard(x)     → supprime x, pas d'erreur si absent
.pop()          → supprime un élément au hasard
.clear()        → vide le set
len(set)        → nombre d'éléments
x in set        → True si x existe
set1 | set2     → union
set1 & set2     → intersection
set1 - set2     → différence
set1 ^ set2     → différence symétrique
```
